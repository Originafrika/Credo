# Credo — Courtier Crédit IA

Vérifie ta solvabilité en 5 minutes. Découvre ton montant exact et l'institution partenaire idéale.

## Concept
Plateforme B2C où l'utilisateur paie ~3,500 FCFA, l'IA analyse ses documents avec RAG sur les règles des partenaires financiers, et lui dit: solvabilité, montant, institution recommandée.
